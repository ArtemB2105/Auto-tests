Примеры тестов на Playwright+pytest+request+allure с использованием фикстур и параметризаций

Тесты на основные методовы Rest API,
объект тестирования и документация - [Fake Store API](https://fakestoreapi.com/).
Архитектура:
Разделение на слои: HTTP-клиент → адаптер → сервис.  
Использование фикстур с teardown.
Pydantic-модели для проверки структуры ответов.
Обёртки над assert с Allure-аттачами при падении.
Faker для генерации тестовых данных.

Авто-тесты на ключевой функционал+позитивные/негативные проверки,
объект тестирования - вебсайт(https://www.saucedemo.com/),
Сайт состоит из окна авторизации, каталога, карточек товаров, окна оформления заказов.
**В ПРОЦЕССЕ**
реализован в стиле Page Object Model
